# MAC0350
Repositório para armazenar exercícios, trabalhos e projetos da disciplina MAC0350 - Introdução ao Desenvolvimento de Sistemas de Software

## Projeto Individual 
O projeto será um sistema de planejamento e gerenciamento de viagens. Nele será possível criar viagens, itinerários, planejar gastos e visualizar seu histórico de viagens e lugares visitados.
